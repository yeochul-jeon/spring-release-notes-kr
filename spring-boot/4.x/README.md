# Spring Boot 4.x 시리즈

Spring Boot 4는 Spring Framework 7을 기반으로 하는 차세대 버전입니다.

---

## 핵심 테마

- **Spring Framework 7 기반**: 최신 Spring Framework의 기능 활용
- **가상 스레드(Virtual Threads)** 기본 지원 강화
- **Spring Boot 3.x에서 Deprecated된 기능 제거**
- **성능 최적화 및 개발자 경험(DX) 개선**

---

## 버전 목록

| 버전 | 릴리즈 | 주요 내용 |
|------|--------|----------|
| [4.0](4.0.md) | 2025-11 | Spring Framework 7 기반, Jakarta EE 11, Jackson 3 |

---

## 요구사항

- **Java**: 17+
- **Spring Framework**: 7.0
- **Jakarta EE**: 9+
- **Gradle**: 8.14+ / 9.x
- **Maven**: 3.6+
